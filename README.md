# RAG-Based AI Assistant

## Overview
A Retrieval-Augmented Generation (RAG) AI Assistant that retrieves information from documents and generates intelligent responses using LLMs and vector embeddings.

## Features
- Document ingestion
- Text chunking
- Embedding generation
- Semantic search
- Question answering
- Retrieval-Augmented Generation (RAG)

## Technologies Used
- Python
- LangChain
- Vector Database
- LLM
- Sentence Transformers

## Project Structure
create_chunks.py
merge_chunks.py
preprocess_json.py
read_chunks.py
process_video.py
video_to_mp3.py
